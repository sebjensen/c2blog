Using the master slide

Seb Jensen

Especially considering that I have created backgrounds with assets from the Rocket League Esports website, I should now look to using the master slide functionality in PowerPoint. With this, I will be able to apply the backgrounds to the homepage and all the content pages that I will need to fill with the assets created and copy that I have worked on in previous sessions.

You can use the root slide in the master slide view to add colours and change the font. This will apply to all slide variations. You can also create your own types of slides, which is what I have done to achieve a different background for my homepage and content pages.

Perspective project approach

The primary strength from this session is that now I will be able to ensure consistency and continuity across all slides that I will create for my interactive kiosk product. This will go a long way to keep a positive and sensical user experience, where all slides will feel familiar and associated with each other and the Rocket League Esports website that I am reimagining.

It would be even better if I added extra features to my template for the content slides. While I have been able to add a background and fonts that will be shared across these slides, it would save more time when putting the assets together if I sorted out the layout by adding shapes and extra text boxes now. While I can still duplicate slides, this will not be as efficient.

My short-term goal that I wish to complete in the next session(s) is to look at how I will be able to incorporate animations into my interactive kiosk product. I also want to refer back to my initial research for the various interactive features I identified. I can then port some or all of these across into a kiosk format.

In the long-term, I will need to consider how I want to set out the homepage and all of the content page. Now I have sorted out the master slide, it will save me from having to sort out the background each time I create a new content slide. I need to think about how I want to combine text with the assets I have collected and created.
